# DevSecOps Platform

A production-style DevSecOps platform built locally using open-source tools.

This repository is designed to demonstrate practical skills in:

- Linux and troubleshooting
- Docker and containerization
- Kubernetes and EKS concepts
- CI/CD using GitHub Actions, GitLab CI/CD, and Jenkins
- Infrastructure as Code using Terraform and Ansible
- Security scanning using Trivy, Gitleaks, Semgrep, Checkov, and OWASP ZAP
- Observability using Prometheus, Grafana, Loki, and Alertmanager
- Incident response, runbooks, and production troubleshooting

## Objective

The goal of this project is to build a realistic DevSecOps platform that can be used for learning, interviews, and portfolio demonstration.

This is not a collection of random labs. Each component is connected to a production engineering workflow.
